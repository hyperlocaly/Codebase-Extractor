Backend foundation is considered stable.

Do not perform another audit.

Objective:
Prepare the system for frontend implementation.

Tasks:

1. Generate a Frontend Build Plan

Analyze all existing API routes, schemas, permissions, and workflows.

Create:

- Screen inventory
- Navigation structure
- User journeys
- Required forms
- Required tables
- Required dashboards
- Required admin pages

Group by:

Consumer
Business Owner
Marketplace Admin
Super Admin

2. Create Frontend Architecture

Using the existing stack:

- React
- TypeScript
- Vite
- React Query
- React Router
- Tailwind
- Shadcn UI

Design:

  pages/
  components/
  features/
  hooks/
  services/
  layouts/
  routes/
  types/

Classify all frontend work into:

Phase A (MVP Critical)
- Authentication
- Marketplace selection
- Business onboarding
- Business profile management
- Products
- Services
- Search
- Business directory

Phase B
- Portfolio
- Reviews
- Verification
- Saved items
- Notifications

Phase C
- Analytics
- Advanced moderation
- Admin operations

For every screen provide:

- Purpose
- Route

- Permissions required
- Components required
- Loading states
- Error states
- Empty states
