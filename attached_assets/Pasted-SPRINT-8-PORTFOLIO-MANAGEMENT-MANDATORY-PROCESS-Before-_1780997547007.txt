SPRINT 8 — PORTFOLIO MANAGEMENT

MANDATORY PROCESS

Before writing any code:

1. Perform a complete audit.
2. Verify existing backend routes, DB tables, OpenAPI schemas, generated hooks, dashboard routes, public profile rendering, and media upload support.
3. Produce an Audit Report listing:

   * Fully Implemented
   * Partially Implemented
   * Missing
   * Defects
4. Do NOT write code until audit is complete.

GOAL

Allow business owners to manage a professional portfolio.

Portfolio is one of the most important conversion features for tailors, fashion designers, fabric sellers, shoemakers, makeup artists, photographers, and all creative businesses.

OWNERS MUST BE ABLE TO:

* Create portfolio collections
* Edit collections
* Delete collections
* Reorder collections
* Upload portfolio images
* Add captions
* Add descriptions
* Add external links
* Set featured images
* Toggle visibility
* Arrange display order

PUBLIC USERS MUST BE ABLE TO:

* View portfolio collections
* Open image galleries
* View captions/descriptions
* Navigate collection images
* View featured work
