DEFAULT LOCATION BEHAVIOR

7. Add marketplace setting:

default_location_template_id

8. When a marketplace has a default location template:

   Example:
   Marketplace Country = Nigeria

   Registration forms must NOT ask the user to select country.

   Users should start selection from the next available level:

   State → LGA → Area

9. Location APIs must expose:

GET /api/v1/marketplaces/{id}/location-config

Response should indicate:

* country
* default location template
* selectable levels
* hidden levels
